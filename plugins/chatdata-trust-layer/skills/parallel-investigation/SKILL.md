# Parallel Investigation

For data-quality or KPI movement investigations, split the work into independent checks:

- counts by date
- counts by segment
- counts by status
- null checks
- min/max date range
- week-over-week comparison
- recent samples
- outlier samples
- daily trend
- freshness and source tie-out

Run independent reads in parallel when the environment allows it. Return a concise synthesis with the strongest explanation, confidence, caveats, and next repair path.
