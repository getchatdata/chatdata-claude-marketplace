# Trial And Privacy

ChatData v1 posture:

- 7-day free trial
- $49/month after trial
- manual license activation until Stripe is added
- local-first by default
- telemetry off unless explicitly enabled
- customer data and project context stay local unless the user chooses to export or publish

Expired trials block value commands but allow status, license activation, and proof export.

Do not claim tamper-proof licensing, public community metrics, or telemetry-backed savings until those systems exist.
